# Security Policy

Twin-Lab is in pre-launch and has not been audited. It handles authentication,
encrypted API keys, and user data — please treat security issues responsibly.

## Reporting a Vulnerability

If you find a security issue, **please do not open a public GitHub issue**.

Email **markus.baier@harwayexperience.com** with:

- A description of the issue
- Steps to reproduce (if applicable)
- The affected version or commit hash

I'll respond as soon as I can. Coordinated disclosure preferred.
